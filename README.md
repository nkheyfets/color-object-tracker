# Color Object Tracker

A real-time computer vision application built with Python and OpenCV that detects and tracks a specified color (blue) object using a webcam.

## Features

- Real-time webcam capture
- Blue object detection using HSV color filtering
- Bounding box around detected object
- Center point tracking
- Left / Center / Right navigation commands
- Distance estimation (Far / Medium / Close)
- Movement trail visualization
- FPS (Frames Per Second) display
- Screenshot capture

## Technologies

- Python
- OpenCV
- NumPy
